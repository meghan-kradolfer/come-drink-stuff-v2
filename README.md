# Come and drink stuff!
- - -

### Installing

Install required packages
``` 
npm install 
```

### Running
Run on local server
``` 
yarn start-dev
```

### Building
``` 
yarn build-dev
```

### Push to origin
``` 
git add .
git commit -m "another commit"
git push origin
```

### Deploying
``` 
git add .
git commit -m "another commit"
git push heroku master

heroku open
```
